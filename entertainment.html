Strand - Fun Commands

These are amusing commands and game modules for entertainment in your server.

This directory assumes that you understand the basics of adding commands. We assume you are using the default command prefix, ?. Commands with inputs are denoted as ?command [input]. Commands with choice inputs are denoted as ?command [input1/input2].

Trigger Use Command Code
?8ball [question] Asks the all knowing Magic 8-Ball a question.
?roulette Plays a game of Russian Roulette, muting the user for 3 mninutes if they lose. The user cannot be a moderator or administrator.
